## 🧠 Topics Covered (Theory)

### 📂 Database Operations

These commands are used to manage databases.

* **CREATE DATABASE** → Used to create a new database
* **DROP DATABASE** → Used to delete an existing database
* **SHOW DATABASES** → Displays all available databases

---

### 📂 Table Operations

Used to create and manage tables inside a database.

* **CREATE TABLE** → Creates a new table with defined columns
* **INSERT INTO** → Adds data into the table
* **ALTER TABLE** → Modifies table structure (add/remove columns)
* **DROP TABLE** → Deletes a table permanently

---

### 📂 Constraints

Constraints are rules applied on table columns to ensure data accuracy and integrity.

* **PRIMARY KEY** → Uniquely identifies each record (no NULL, no duplicate)
* **FOREIGN KEY** → Links one table with another (maintains relationships)
* **NOT NULL** → Ensures column cannot have NULL values
* **UNIQUE** → Ensures all values in a column are different
* **DEFAULT** → Assigns a default value if none is provided
* **CHECK** → Ensures a condition is satisfied before inserting data

---

### 📂 Data Retrieval

Used to fetch and filter data from tables.

* **SELECT** → Retrieves data from a table
* **WHERE** → Filters records based on conditions
* **DISTINCT** → Removes duplicate values
* **ORDER BY** → Sorts data (ascending/descending)
* **LIMIT** → Restricts number of output rows

---

### 📂 Aggregate Functions

Used to perform calculations on multiple rows.

* **COUNT()** → Counts number of rows
* **SUM()** → Adds values
* **AVG()** → Finds average
* **MAX()** → Finds highest value
* **MIN()** → Finds lowest value

---

### 📂 Grouping

Used to group rows that have similar values.

* **GROUP BY** → Groups rows based on a column
* **HAVING** → Filters grouped data (used with aggregate functions)

👉 Difference:

* WHERE → filters rows before grouping
* HAVING → filters groups after grouping

---

### 📂 Joins

Used to combine data from multiple tables.

* **INNER JOIN** → Returns matching records from both tables
* **LEFT JOIN** → Returns all records from left table + matched from right
* **RIGHT JOIN** → Returns all records from right table + matched from left
* **FULL JOIN** → Returns all records from both tables (using UNION in MySQL)
